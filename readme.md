# Student Project Progress & Performance Tracking System

## Problem Statement
Tracking student project progress and individual contributions is often manual and inefficient.

## Solution
This project provides a web-based system to manage student projects, track progress, and analyze GitHub contributions.

## Features
- User authentication (Student / Mentor / Admin)
- Project and milestone management
- Progress tracking dashboard
- GitHub repository integration

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python Flask
- Database: SQLite
- GitHub API

## How to Run
1. Clone repository
2. Run backend server
3. Open frontend in browser

## Future Scope
- Advanced GitHub analytics
- Notifications
- Performance scoring
